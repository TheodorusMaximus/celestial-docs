Dictionary of Terms

Agile

A project management methodology that emphasizes delivering functional software in short time cycles, working collaboratively with the team and responding quickly to changes.

  • Usage examples: Our project is using Agile methodology to manage the software development.
  • Reference sources: Scrum.org
  • Abbreviations: None.
  • Synonyms: Agile methodology.
  • Contextualization: Agile is an iterative and incremental approach to project management that prioritizes delivering value to the customer continuously. It is commonly used in software development projects.

Back-end

The part of a software application that handles data storage, processing, and server-side logic.

  • Usage examples: Our back-end team is responsible for developing the application's data storage and processing functionality.
  • Reference sources: Techopedia
  • Abbreviations: None.
  • Synonyms: Server-side.
  • Contextualization: The back-end is the core of a software application that handles complex data processing and business logic, interacting with the front-end to provide a seamless user experience.

Front-end

The part of a software application that users interact with directly, including the user interface and user experience.

  • Usage examples: Our front-end team is responsible for designing and developing the user interface and user experience of the application.
  • Reference sources: Techopedia
  • Abbreviations: None.
  • Synonyms: Client-side.
  • Contextualization: The front-end is the user-facing part of a software application, responsible for delivering a seamless and intuitive user experience. It interacts with the back-end to retrieve and display data.

API

Application Programming Interface

A set of protocols, tools, and standards for building software applications and enabling communication between different software systems.

  • Usage examples: Our application uses an API to communicate with external systems and services.
  • Reference sources: Webopedia
  • Abbreviations: None.
  • Synonyms: None.
  • Contextualization: An API provides a standardized way for different software systems to communicate with each other and exchange data, enabling the creation of complex applications that integrate with multiple systems.

MVC

Model-View-Controller

A software design pattern that separates the application's data model, user interface, and control logic into separate components.

  • Usage examples: Our application is designed using the MVC architecture, with the data model, user interface, and control logic in separate components.
  • Reference sources: Techopedia
  • Abbreviations: None.
  • Synonyms: None.
  • Contextualization: The MVC pattern is widely used in software development to separate the concerns of data management, user interface, and control logic, making it easier to maintain and update applications.

DevOps

A set of practices that combines software development (Dev) and IT operations (Ops) to shorten the development cycle, improve software quality, and reduce time to market.

  • Usage examples: Our team uses DevOps practices to automate the deployment process and improve the reliability of our software.
  • Reference sources: Microsoft Azure
  • Abbreviations: None.
  • Synonyms: None.
  • Contextualization: DevOps is a cultural shift in software development.
